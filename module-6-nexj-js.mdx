# Fetching - preferching on server component

1. Next server виконує компонент page - app/notes/page.tsx
2. Виконується запис з queryClient.prefetchQuery
3. Результат запиту зберігається у кеші queryClient

# Use prefethced chache for client component

4. Виконується компонент NoteListPage
5. Отримує даннні від queryClient
6. В браузер відправляється сторінка
7. В браузері ініціалізується useQuery
8. useQuery вже використовує кеш queryClient ["notes"]
9. refetchOnMount: false щоб не робити зайвий запит тому що
